My Certifications

Education
MEAN STACK DEVELOPMENT(2020)

MEAN stack development refers to a collection of technologies used for building web applications. The acronym MEAN stands for:
M - MongoDB: A NoSQL database system for storing data in document form.
E - Express: A server-side framework for Node.js, used for building web applications.
A- Angular: A JavaScript-based open-source front-end web application framework.
N - Node.js: A server-side platform built on Chrome's JavaScript runtime for easily building fast and scalable network applications.
The MEAN stack provides developers with a complete end-to-end solution for building dynamic web applications, using only JavaScript. The combination of MongoDB, Express, Angular, and Node.js provides a full-stack JavaScript environment that enables developers to write code for both the front-end and back-end of a web application using a single programming language.
